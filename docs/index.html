时刻音乐

利用Vue+Webpack+Nginx构建的在线音乐Web App.

免安装

利用浏览器打开,可直接点击添加至桌面,接着你会在桌面发现时刻音乐的图标,这一切就像你添加小程序一样简单.

随时用

你可以点击已经添加至桌面的图标,或者通过扫描二维码的方式在任何平台使用时刻音乐.

响应快

所有美好的事情都会在5s内展现在你的眼前,你可以体验近乎接近原生App的精彩世界.

技术

Vue、Vue-Router、VueX

包括Vue-Router、VueX、Vue-Cli

Node.js

利用伪造请求头,爬取网易云音乐接口,封装为Node API

Webpack、Rollup

利用Webpack进行项目打包,利用rollup进行模块打包

ES6

均采用ES6进行开发

Nginx

利用Nginx作为反向代理服务器

ESC云服务器

采用阿里云ECS云服务器

预览

一起来体验一下吧

用任何可以扫描二维码的平台扫描右方的二维码,即可使用时刻音乐🎵
